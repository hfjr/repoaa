个人征信查询及信息使用授权书


融桥宝及 （合作金融机构等）：

鉴于本人作为借款人通过融桥宝的居间服务向出借人借款，依据《征信业管理条例》、《个人信用信息基础数据库管理暂行办法》、《中国人民银行征信中心关于印发企业及个人信用报告查询和异议处理业务规程的通知》等相关制度的规定，本人同意并不可撤销地授权融桥宝及合作伙伴：

1、向包括但不限于信息基础数据库、中国人民银行批准的征信机构、公安系统等查询本人的信用报告或记录；

2、将本人的身份识别、职业和居住地址等个人基本信息以及在上述业务和担保等信用活动中形成的交易记录和个人信贷信息，提供给包括但不限于信息基础数据库、中国人民银行批准的征信机构、公安系统等信用信息基础数据库，以及其他根据法律法规要求所做的信息披露；

3、使用本人在融桥宝合作伙伴的数据信息用于借款的审核、贷后管理及催收等用途。本授权委托书自签发之日起至上述借款业务结束(所有相关款项全部结清)之日止有效。本人完全知晓并理解，即使借款申请未被审核通过，本文件各项授权的有效性不受影响。

本人确认通过融桥宝平台线上签署本授权委托书。
