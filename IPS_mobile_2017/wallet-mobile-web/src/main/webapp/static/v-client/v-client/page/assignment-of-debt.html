3.受让人在此知悉并确认：转让人对借款人不承担任何保证责任，若出现借款人未能按期还款的，应当由受让人依照《借款合同》的约定向借款人要求赔偿。

七.信用保证保险

1.受让人与转让人在此知悉并确认，借款人与转让人于签署《借款合同》的同时，已向融桥宝指定的第三方合作保险公司（以下简称“保险公司”）投保保证保险产品，被保险人为转让人。转让人确认所转让债权受保险公司的保证保险保障。在债权转让合同生效时，转让人同时向受让人转让债权所对应的保险权益，且在出现理赔时，配合受让人向保险公司提出索赔请求，完成理赔程序，并按约支付赔款。

2.受让人与转让人均在此确认且无条件、不可撤销地授权融桥宝有权代为行使被保险人的相关权利，包括但不限于基于上述第七条第一款的有效投保协议及相关法律法规的要求，向保险公司提出索赔请求、签署索赔函、收集与递交理赔所需单证、身份证明材料、确认保险赔款及代为收取赔款、签发权益转让书等。保险公司就赔款事宜与融桥宝达成一致并按约支付赔款的，即视为保险公司已完成履行《投保协议》项下之所有义务，受让人不得再向保险公司提出任何索赔请求。受让人同意上述授权溯及融桥宝既往已实际行使的与本合同相关的前述权利，转让人对此亦不持异议。

3.受让人同意并确认，借款人逾期支付本金及/或利息时，受让人授权融桥宝向保险公司告知逾期情况，保险公司根据《投保协议》的约定，及上述第七条第一款的规定向债权受让人进行赔付。

4.双方在此确认：在保险公司根据《投保协议》的约定，及上述第七条第一款的规定向受让人赔付本合同项下的借款本金和利息时，受让人将对于借款人的全部权益（包括追究借款人违约责任的权利）转让给保险公司，并在保险公司赔付前，按保险公司要求出具《权益转让书》、身份证复印件及相关证明文件，否则保险公司有权延迟赔付。保险事故发生后，双方承诺不会单方免除借款人的还款义务，否则保险公司有权拒绝赔付。

5.双方同意，赔付等待期（如有）内无利息。此条款所指赔付等待期是指保险公司受理融桥宝的理赔申请后至实际赔付受让人之日的期间。

八.双方陈述与保证

1.转让人的陈述与保证：

1) 转让人保证其提供的信息和资料均为真实、完整、有效。

2) 转让人确认，转让人转让的债权是其合法有效的债权，不存在转让的限制，符合融桥宝关于债权转让的条件。

2.受让人的陈述与保证

1) 受让人保证其提供的信息和资料均为真实、完整、有效，并且受让让人进一步保证其不会提供任何虚假信息或隐瞒重要事实。

2) 受让人保证本次所支付的受让标的债权的资金来源合法合规，受让人进一步承诺此次资金支付符合法律法规的相关规定，且不存在任何洗钱等违法违规行为。

3) 受让人保证所支付的资金属于受让人有权支配的资金，且不存在妨碍受让人参与本次债权转让的情形。如果第三人对资金归属、合法性问题发生争议的，由受让人自行解决，与融桥宝无关。若因此给融桥宝造成损失的，受让人应赔偿一切损失。

九.保密条款

1.基于本次转让所获得的包括本合同在内的相关信息或商业约定，任何一方不得向本合同主体及融桥宝以外的其他方透露，根据法律法规及有关监管机构要求应当进行披露的，或为债权再转让之目的进行披露的除外。

2.各方保证：为履行本合同各自委托的专业机构和人员承担上述保密义务。

3.任何一方违反保密义务，守约方有权要求违约方赔偿损失。

十.违约责任

1.本合同双方均应履行本合同所约定的义务。任何一方不履行或不完全履行本合同所约定义务的，违约方应承担违约责任并赔偿对方因此而产生的费用和损失，包括但不限于直接损失、间接损失、实现债权费用、诉讼费、律师费等。

1.双方在此确认，双方均己仔细阅读了本合同及附件《标的债权明细》的条款内容，对本合同及标的债权的所有内容均已充分了解且无异议，并且对本合同项下的有关权利、义务和责任已充分理解。

3.转让人与受让人一致确认，融桥宝仅为本合同项下的债权转让交易提供中介服务，并不就本次债权转让交易向任何一方承担任何保证责任或其他担保责任。

4.转让人和受让人均同意并确认，转让人、受让人通过融桥宝平台所采取的任何行为所产生的法律效果和法律责任均分别归属于转让人、受让人本人；并且转让人、受让人授权和委托融桥宝根据本合同所采取的全部行动和措施的法律后果均分别归属于转让人、受让人本人，与融桥宝无关，融桥宝财也不因此承担任何责任。

5.转让人与受让人确认并同意，一致委托融桥宝对本合同项下的任何金额进行计算并允许通过融桥宝平台发布；在无明显错误的情况下，融桥宝对本合同项下任何金额的任何证明或计算结果，应作为该金额有关事项的终局证明。

6.受让人同意和确认，由于债权受让人比较分散，无法亲自管理和行使对借款人的权利，为保障债权受让人的债权安全，债权受让人将其在本合同项下对借款人的权利不可撤销地授权并委托融桥宝代为行使，融桥宝接受委托行使上述对借款人的权利所产生的法律责任及后果由债权受让人承担。
