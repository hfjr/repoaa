主要风险说明如下：

一、政策风险

因国家宏观政策和相关法律法规发生变化，影响融桥宝平台系统功能服务的正常提供。对此融桥宝平台不承担任何责任。

二、市场风险

国家宏观政策、货币政策、行业政策、资金市场供求关系的变化以及整体经济形势的变化，可能会对利率、汇率、资金成本和商品价格等因素产生负面影响，从而不利于投资人获得预期收益，甚至造成投资人投资本金的损失。对此融桥宝平台不承担任何责任。

三、信用风险

借款方因突发事件导致短期或者长期丧失还款能力，或者借款方的还款意愿发生变化的，可能导致不能按时向投资人偿还本金和收益。如果投资人的投资标的由第三方提供相关保障，届时，投资款项将根据融桥宝平台相关公告和规则得到赔付或者补偿，投资人对于不足部分资金的回收将可能延期。

四、操作风险

投资人在投资过程中妥善保管相关账户，因投资人的账号及密码信息被盗或泄露的，或其他投资人操作不当的原因，可能导致投资人身份被仿冒，因此造成一定的资金损失。对此融桥宝平台不承担任何责任。

五、不可抗力风险

由于自然灾害、战争、法律法规或政策等无法避免或无法控制的因素的出现，从而导致投资人无法获得投资收益乃至本金遭受损失，甚至影响投资标的偿付等的正常进行。对此融桥宝平台不承担任何责任。

六、其他风险
